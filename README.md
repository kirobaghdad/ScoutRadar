# ScoutRadar - Data Science Project (Phase 2)

## Setup Instructions

This project uses `poetry` for dependency management.

1. **Install Poetry**: If you haven't already, install poetry.
2. **Install Dependencies**: Run `poetry install` in the root folder.
3. **Activate Environment**: Run `poetry shell` to activate the virtual environment, or prefix commands with `poetry run`.

## Folder Structure

- `data/` : Dataset files
- `docs/` : Documentation and PDFs
- `notebooks/` : Jupyter Notebooks for initial exploration
- `src/` : Main source code
- `tests/` : Pytest tests
- `models/` : Saved models
